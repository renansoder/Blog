## Projeto de um BLOG de conteúdos escritos diversos.

Blog particular para criação de conteúdos diversos. Apresentação pública de artigos.

Projeto completo (CRUD), utilizando:

Backend:

- Node.js
- MySql

Frontend:

- Bootstrap
- TinyMCE

Pacotes utilizados:

- bcryptjs
- dotenv
- ejs
- express
- express-session
- mysql2
- sequelize
- slugify
- nodemon
